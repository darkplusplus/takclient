# dpp.takclient

`dpp.takclient` is a .NET TAK client library built on top of [`dpp.cot`](https://www.nuget.org/packages/dpp.cot).

It provides:

- TAK XML streaming connections
- TAK protobuf streaming connections
- automatic protobuf negotiation over the initial XML session
- parsed inbound CoT message delivery

Targets:

- `net6.0`
- `net8.0`
- `net10.0`

Source:

- <https://github.com/darkplusplus/takclient>
